# QMK Toolbox

View on GitHub (opens new window)

This is a collection of flashing tools packaged into one app. It supports auto-detection and auto-flashing of firmware to keyboards.

# Flashing

QMK Toolbox supports the following bootloaders:

  • Atmel/LUFA/QMK DFU via dfu-programmer (opens new window)
  • Caterina (Arduino, Pro Micro) via avrdude (opens new window)
  • Halfkay (Teensy, Ergodox EZ) via Teensy Loader (opens new window)
  • ARM DFU (STM32, Kiibohd) via dfu-util (opens new window)
  • Atmel SAM-BA (Massdrop) via Massdrop Loader (opens new window)
  • BootloadHID (Atmel, PS2AVRGB) via bootloadHID (opens new window)

And the following ISP flashers:

  • USBTiny (AVR Pocket)
  • AVRISP (Arduino ISP)
  • USBasp (AVR ISP)

If there's an interest in any others, they can be added if their commands are known.

# HID Console

The Toolbox also listens to HID messages on usage page 0xFF31 and usage 0x0074, compatible with PJRC's hid_listen (opens new window).

# Installation

# Dependencies

When using the QMK Toolbox on Windows, it will prompt at first run to install the necessary drivers. You can get the latest release of the QMK Driver Installer here (opens new window).

# Download

A Windows and macOS version are available, and you can get the latest release here (opens new window).

For Homebrew users, it is also available as a Cask:

$ brew tap homebrew/cask-drivers
$ brew cask install qmk-toolbox
